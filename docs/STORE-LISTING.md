## Detailed description
```
Quillery is a fast, private home for all your AI prompts — and it opens in the
browser side panel, so it stays open beside ChatGPT, Claude, Gemini and anything
else you're working in.

✦ ORGANIZE
• Save prompts with titles, categories, tags and ⭐ ratings
• Custom categories with your own emoji and colour
• Fuzzy search, smart sorting (most used, rating, recent, A–Z, pinned)
• Pin favourites and copy them with one click

✦ REUSE
• {{variables}} — fill in values with a live preview, then copy the result
• One-click copy to your clipboard
• Import / export everything as JSON or Markdown

✦ OPTIMIZE (free, on-device)
• Rewrite any prompt for ChatGPT, Claude or Gemini using proven, model-specific
  best practices — all computed locally. No account, no API key, nothing sent
  anywhere.

✦ PRIVATE BY DESIGN
• All data stays in your browser's local storage — no servers, no accounts, no
  tracking, no analytics. The only permission requested is "sidePanel".

Quillery is also available as a desktop app.
```
